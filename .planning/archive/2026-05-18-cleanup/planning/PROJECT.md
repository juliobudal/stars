# Project: LittleStars 🌟

## Overview
LittleStars is a gamified task management application for families, focused on financial education for children through a "star economy".

## Tech Stack
- **Framework:** Ruby on Rails 8.1
- **Asset Pipeline:** Vite + Propshaft
- **Frontend:** Tailwind CSS 4.0, Stimulus, ViewComponent
- **Database:** PostgreSQL
- **Real-time:** Turbo Streams (ActionCable)
- **Infrastructure:** Docker + Kamal

## Milestones
- [x] **Milestone 1: MVP Core** (Completed 2026-04-19) - Basic CRUD, Approval Flow, Kid View, Parent Dashboard.
- [/] **Milestone 2: UI/UX Duolingo Rebranding** (In Progress) - Overhaul of the entire interface to match Duolingo's bouncy, 3D, and vibrant style.
