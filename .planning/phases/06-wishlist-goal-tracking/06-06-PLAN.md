    ```erb
    <% pinned_here = current_profile.wishlist_reward_id == reward.id %>
    <% if pinned_here %>
      <%= button_to kid_wishlist_path,
                    method: :delete,
                    form: { class: "absolute top-2 right-2 z-10", data: { turbo: true } },
                    class: "ls-btn-3d inline-flex items-center gap-1 px-2 py-1 rounded-[10px] text-[11px] font-extrabold",
                    style: "background: var(--star-soft); color: var(--c-amber-dark); box-shadow: 0 3px 0 var(--star-2);",
                    aria: { label: "Remover meta" } do %>
        <%= render Ui::Icon::Component.new(:star, size: 12, color: "var(--c-amber-dark)") %>
        Meta
      <% end %>
    <% else %>
      <%= button_to kid_wishlist_path,
                    method: :post,
                    params: { reward_id: reward.id },
                    form: { class: "absolute top-2 right-2 z-10", data: { turbo: true } },
                    class: "ls-btn-3d inline-flex items-center gap-1 px-2 py-1 rounded-[10px] text-[11px] font-bold",
                    style: "background: var(--surface); color: var(--text-muted); border: 2px solid var(--hairline); box-shadow: 0 3px 0 var(--hairline);",
                    aria: { label: "Definir como meta" } do %>
        <%= render Ui::Icon::Component.new(:star, size: 12, color: "var(--text-muted)") %>
        Meta
      <% end %>
    <% end %>
    ```

    Critical:
    - The button is positioned `absolute top-2 right-2 z-10` — outside the redeem modal trigger area to prevent click conflict.
    - Both branches use `aria-label` for the screen reader (the visible text "Meta" is short; aria provides the action verb).
    - Use `var(--star-soft)`, `var(--c-amber-dark)`, `var(--star-2)`, `var(--surface)`, `var(--text-muted)`, `var(--hairline)` — all from theme.css. NO raw hex.
    - `ls-btn-3d` provides press effect + reduced-motion (CLAUDE.md C-7).
    - `data: { turbo: true }` ensures Turbo handles the form submission (default in Rails 8 but explicit for clarity).
    - The pinned variant uses the warm amber palette (Duolingo "star" tokens) to communicate "this is your goal." Unpinned uses neutral/ghost styling.
    - Make sure the outer card wrapper `<div>` has `class` containing `relative` (or add `relative` to the existing class list) — required for `absolute top-2 right-2` to position correctly. Check the existing class list first; if `relative` is already there, no change needed.
