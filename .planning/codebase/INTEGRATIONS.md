# Dependencies and Integrations

## Active Integrations
- **PostgreSQL**: Primary relational database, handling all application state and activity logs. Configured via `docker-compose.yml`.
- **JetRockets UI**: External UI component library integration providing ready-to-use ViewComponents.

## Build and Asset Integrations
- **Vite & Node.js**: Integrated for building frontend assets (Stimulus controllers, TailwindCSS).

## Future/Potential External Integrations
- Currently, the application operates as a self-contained system. However, any future integrations (e.g., push notifications, email services) will be documented here.
