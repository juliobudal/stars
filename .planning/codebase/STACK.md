# Technology Stack

## Core Technologies
- **Framework**: Ruby on Rails 8.x (Fullstack)
- **Language**: Ruby 3.3+
- **Database**: PostgreSQL 16+

## Frontend
- **UI Components**: JetRockets UI (latest)
- **View Layer**: ViewComponent + ERB
- **CSS**: TailwindCSS 4.0
- **JavaScript**: Stimulus 3.x
- **Navigation & Real-time**: Turbo 8.x (Drive, Frames, Streams)
- **Build Tool**: Vite (via `vite_rails`)

## Infrastructure
- **Containerization**: Docker + Docker Compose
- **Development Environment**: Devcontainers
- **Servers**: Kamal (configured, but mostly focuses on MVP)

## Testing
- **Framework**: RSpec
- **Fixtures/Mocks**: FactoryBot + Faker
- **E2E/System**: Capybara + Selenium WebDriver
- **Linters**: Standard (ruby) + Rubocop
